| Эндпоинт         | Метод | Что делает                                                                                     |
| ---------------- | ----- | ---------------------------------------------------------------------------------------------- |
| /                | GET   | Проверка, что сервер запущен и работает.                                                       |
| /predict         | POST  | Вычисление тональности одного текста через загруженную нейросеть.                              |
| /train/start     | POST  | Запуск обучения модели (в отдельном процессе). Поддерживает обязательный/форсированный запуск. |
| /train/status    | GET   | Получение текущего статуса обучения: идёт ли обучение, PID, прогресс, логи.                    |
| /system/health   | GET   | Быстрый health-check сервера и проверка наличия модели/словари.                                |
| /system/info     | GET   | Информационный эндпоинт: версии, пути модели, время последнего обучения, окружение.            |
| /dataset/analyze | POST  | Вычисление тональности датасета отзывов через загруженную нейросеть.                           |

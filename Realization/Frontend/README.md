# Sentiment Analysis Frontend

Фронтенд приложение для анализа тональности текста на Vue 3 с Element Plus.

## Установка

```bash
npm install
```

## Запуск dev сервера

```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:5173

## Сборка для production

```bash
npm run build
```

## Требования

- Node.js 16+
- npm или yarn
- Backend сервер должен быть запущен на http://localhost:8000
